# TrendPulse Project

This project fetches trending data from an API, cleans it, analyzes it, and visualizes insights using Python.

## 📂 Project Structure

* src/ → Python scripts for each task
* outputs/ → Generated charts and dashboard

## 🧠 Tasks Completed

1. Data Collection (API)
2. Data Cleaning (Pandas)
3. Data Analysis (NumPy)
4. Data Visualization (Matplotlib)

## 📊 Outputs

* Top Stories Chart
* Category Distribution
* Score vs Comments Scatter Plot
* Combined Dashboard

## 🚀 Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
